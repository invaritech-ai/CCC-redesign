## Data Entry Tips

1. **Slugs**: Most content types have auto-generated slugs. You can customize them if needed, but the default (based on title) usually works well.

2. **Rich Text Fields**: Fields marked as "array of blocks" support rich text formatting. Use the toolbar to add headings, bold, lists, links, etc.

3. **Images**: 
   - Always add descriptive alt text for accessibility
   - Use high-quality images (recommended: 1200px width minimum)
   - For featured content, use compelling images that represent the content well

4. **Featured Flags**: Use the "featured" checkbox to highlight important content that should appear prominently on the website.

5. **Display Order**: For content types with display order (team members, partners, FAQs), lower numbers appear first. Use increments of 10 (1, 10, 20, etc.) to make reordering easier.

6. **Dates**: 
   - Use the date picker for date fields
   - For datetime fields, include both date and time
   - Display time is optional and allows custom formatting

7. **Required Fields**: Fields marked with validation rules are required. Make sure to fill these in before publishing.

8. **Categories and Tags**: 
   - Categories are predefined options - select from the list
   - Tags are free-form - add relevant keywords

9. **File Uploads**: 
   - PDFs should be optimized (under 5MB when possible)
   - Images should be in JPG or PNG format
   - Videos should be uploaded to a hosting service and linked via URL

10. **Content Quality**: 
    - Write clear, engaging content
    - Proofread before publishing
    - Use consistent formatting and style
    - Include relevant keywords for SEO
